Типы данных

В JavaScript есть несколько типов данных:

  • 1.Строки (String)используются для хранения текстовых значений. Например, "привет" или 'hello'.
  • 2.Числа (Number): используются для хранения числовых значений. Например, 10 или 3.14.
  • 3.Булевы значения (Boolean): используются для хранения логических значений true (истина) или false (ложь)
    • .
  • 4.Объекты (Object):используются для хранения набора свойств и методов. Например, {name: "John", age: 25}.
  • 5.Массивы (Array): используются для хранения упорядоченных коллекций значений. Например, [1, 2, 3, 4]
    • .
  • 6. Null: используется для указания на отсутствие значения.
  • 7. Undefined: используется, когда переменная была объявлена, но не имеет значения.
  • 8. Символы (Symbol): используются для создания уникальных идентификаторов. Введены в ECMAScript 6.
